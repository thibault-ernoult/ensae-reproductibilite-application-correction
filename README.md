# Titanic

## Instructions

### Création d'un fichier de configuration des paramètres

> [!IMPORTANT]
> Pour que le script `titanic.py` fonctionne, il vous faut créer un fichier `config.yaml`,
de la forme :
```
jeton_api: ####
train_path: ####
test_path: ####
test_fraction: ####
```
Remplacez les #### par les valeurs de votre choix
